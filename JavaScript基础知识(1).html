JavaScript 基础知识(1)

本文档介绍 JavaScript 中的数据类型及其基本示例。

一、JavaScript 数据类型

JavaScript 数据类型分为两大类：原始类型（primitive types）和对象类型（object types）。

  • 原始类型包括：字符串（String）、数字（Number）、布尔值（Boolean）、null、undefined 和 Symbol。
  • 对象类型包括：对象（Object）、数组（Array）和函数（Function）。

二、数据类型示例

1. 字符串类型（String）

let str = "Hello, World!"; // 声明一个字符串变量str，并赋值为"Hello, World!"
console.log(str); // 输出变量str的值，结果是"Hello, World!"

2. 数字类型（Number）

let num = 42; // 声明一个数字变量num，并赋值为42
console.log(num); // 输出变量num的值，结果是42

3. 布尔值类型（Boolean）

let isJavaScriptFun = true; // 声明一个布尔变量isJavaScriptFun，并赋值为true
console.log(isJavaScriptFun); // 输出变量isJavaScriptFun的值，结果是true

4. null 类型

let emptyValue = null; // 声明一个变量emptyValue，并赋值为null，表示没有值
console.log(emptyValue); // 输出变量emptyValue的值，结果是null

5. undefined 类型

let undefinedValue; // 声明一个变量undefinedValue，但没有赋值，默认值为undefined
console.log(undefinedValue); // 输出变量undefinedValue的值，结果是undefined

6. 对象类型（Object）

let person = {
  name: "Alice",
  age: 25
}; // 声明一个对象person，包含name和age两个属性
console.log(person); // 输出对象person的值

let arr = [1, 2, 3]; // 声明一个数组arr
console.log(arr); // 输出数组arr的值

function greet() {
  return "Hello!";
} // 声明一个函数greet
console.log(greet); // 输出函数greet
提示：使用 typeof 可以检测变量的数据类型，例如 typeof "hello" 返回 "string"。
