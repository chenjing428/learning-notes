概述

JavaScript 是一门动态类型、解释执行的脚本语言，主要用于网页交互，也可以在服务器端（Node.js）运行。它的核心语法简单，但灵活性极高。

  • 代码区分大小写
  • 语句以分号 ; 结尾（可省略，但建议保留）
  • 使用 // 写单行注释，/* */ 写多行注释

变量的定义

JavaScript 有三种声明变量的关键字：

关键字作用域能否重复声明是否提升
var函数级能是（初始化为 undefined）
let块级不能是（暂时性死区）
const块级不能是（必须初始化，不能重新赋值）
let name = "Alice";
const age = 25;
var isStudent = true;
现代开发中，优先使用 const，需要重新赋值时再用 let，尽量避免 var。

数据类型

JavaScript 有 7 种原始类型和 1 种引用类型：

原始类型（Primitive）

类型说明示例
string字符串"hello"
number数字（整数、浮点数、NaN、Infinity）42, 3.14
boolean布尔值true, false
undefined未定义undefined
null空值（表示"没有"）null
symbol唯一标识符（ES6）Symbol('id')
bigint大整数（ES2020）123n

引用类型（Reference）

object 对象，包括普通对象、数组、函数、日期等。

let obj = { name: "Alice", age: 25 };
let arr = [1, 2, 3];
let fn = function() {};
用 typeof null 会返回 "object"，这是历史遗留 bug。

表达式与流程控制

常用运算符

  • 算术：+ - * / % **
  • 比较：== === != !== > < >= <=
  • 逻辑：&& || !
  • 赋值：= += -= *= /=
=== 和 !== 比较值和类型，建议始终使用，避免隐式转换陷阱。

条件语句

if (score >= 90) {
  console.log("A");
} else if (score >= 60) {
  console.log("B");
} else {
  console.log("C");
}

// 三元运算符
let result = age >= 18 ? "成年" : "未成年";

循环语句

for (let i = 0; i < 5; i++) {
  console.log(i);
}

let j = 0;
while (j < 5) {
  console.log(j);
  j++;
}

const fruits = ["apple", "banana"];
for (const fruit of fruits) {
  console.log(fruit);
}

函数

函数是 JavaScript 的一等公民，有三种常见定义方式：

1. 函数声明

function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Alice"));

2. 函数表达式

const add = function(a, b) {
  return a + b;
};

3. 箭头函数（ES6）

const multiply = (a, b) => a * b;

const square = x => x * x;

// 多行需要写 return
const sum = (a, b) => {
  const result = a + b;
  return result;
};

参数相关

  • 参数少于定义：多余参数为 undefined
  • 参数多于定义：可用 arguments 对象或剩余参数 ...args 获取
  • 可设置默认值：function greet(name = "Guest") {}
